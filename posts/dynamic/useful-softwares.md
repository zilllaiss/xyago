## Personally used on daily basis.

Softwares that I regularly. A must have for me.

KDE Plasma
: Linux desktop environment which you can costumize to virtually whatever you want.

Konsole
: KDE Plasma default terminal and the best overall. It supports non-latin characters gracefully.

Firefox
: Does not everyone know this browser? 
Although, I actually use LibreWolf since it has sane defaults, but it's still firefox in the end, right?

Sleek
: A Todo app which built on top todo.txt. 
A GUI is more convenient for this, but I also need the todo list to be portable,
so I can open the files without an app as they are just a plain text.

Drawy
: Drawing app which I open whenever I need to quickly dump an idea that appears on my head. 
Contains some nice features beside the basic ones. But more importantly,
it looks nice and fast. 

Neovim
: My main text editor. Normally for coding, but since I installed telescope as one of the plugins,
I also use it for file search and navigation.

Vim
: Same idea as Drawy; mainly used whenever I need to dump any idea that appears on my mind,
but this time with text.

Kwrite
: Another text editor. I use this whenever I think it's more convient to use a GUI. 
It has Vim mode so that's nice.

Yazi
: File explorer for the terminal. Has file previews and good file navigation.

Dolphin
: KDE default file explorer. 
I use it as frequent as Yazi since Dolphin has more features than Yazi.
Actually, Dolphin is arguably the file explorer which has the most features on Linux.

Localsend
: For sending stuff between my devices, and also other people since it is cross-platforms.

Krita
: A graphic editor made by KDE. Mainly used for drawing, but it's also pretty good
for photo editing, vector graphic, and 2D animation.

Qur'an Companion
: Qur'an desktop app. Probably the best Qur'an app that can be used offline on Linux.

Quod libet
: Playlist-based audio player. I chose this over other audio players because of three reasons:
Has all basic features needed for playing audios, has stability, and most importantly: audio waveform.

Mpv
: Has the best plugin system from any video players.
I use it primarily because it has the `thumbfast` plugin, for video seek preview.

Anki
: The word means memorization. 
Just as the name suggest, this is primarily used by people for memorization, 
language learning and reviewing.
But in reality, you can use this for any dictionary memorization.

FreeFileSync
: For doing back up, which is available for Linux and Windows.

Discord
: For chatting with people.

## Helpful to have.

Softwares that must be installed on every computer I have.

LibreOffice
: Office suite. I edit texts on Neovim before moving it to LibreOffice writer.

AppManager
: For managing and "installing" AppImages.

Haruna
: Video player I used before moving to MPV... and actually, it built on MPV as well. 
If there was `thumbfast` installed on it, I would not have moved.
I still have it installed just in case. 

AB Download Manager
: Download manager I need if the file I download has a big size.
